A batch camera system that permits large camera sizes with colored output.
This project demonstrates Basic physics interaction between a controlled entity and the map.

This engine uses Radish for Non-Blocking input. See [https://github.com/thelowsunoverthemoon/radish]

This project is ongoing, with a fully featured batch game engine the end goal, where game rules are directly driven by map content.

Note: The python mapmaker script is unfinished.
